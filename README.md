# Rouky - Devenez Ingénieur IA étape par étape

**Rouky** est une application web moderne, interactive et gamifiée conçue pour vous aider à maîtriser tous les concepts clés de l'ingénierie IA.

## 🚀 Fonctionnalités

- **18 phases** structurées couvrant tous les aspects de l'ingénierie IA
- **200+ tâches** à accomplir
- **54 projets** pratiques avec différents niveaux de difficulté
- Système de **XP, niveaux, streaks et badges**
- **Dashboard** avec statistiques détaillées
- **Suivi de progression** complet
- **Export/Import** de la progression
- Design **responsive** et **thème sombre**

## 🛠 Stack Technique

- Next.js 14 + App Router
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL
- Vercel
- lucide-react
- Recharts
- date-fns

## 📦 Installation

```bash
npm install
cp .env.example .env
# Configurer DATABASE_URL
npx prisma generate
npx prisma db push
npm run dev
```

## 🚀 Déploiement Vercel

1. Connecter le dépôt GitHub
2. Créer Vercel Postgres
3. Ajouter DATABASE_URL
4. Déployer
